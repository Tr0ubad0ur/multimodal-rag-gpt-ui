# Frontend

Интерфейс для дипломного проекта:
мультимодальный поиск и анализ данных на базе RAG + GPT.

## Stack

- Nuxt 4
- Vue 3
- TypeScript
- TailwindCSS

## Run

```bash
cd frontend
npm install
npm run dev
```

По умолчанию frontend ожидает backend на `http://localhost:8000`.

## Env

```bash
NUXT_PUBLIC_API_BASE=http://localhost:8000
```
